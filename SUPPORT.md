# Support

## Getting Help

### Documentation

Start with the documentation:

- [Introduction](docs/introduction.md)
- [FAQ](docs/faq.md)
- [Troubleshooting Guide](docs/troubleshooting.md)

### Issues

If you've found a bug or have a feature request:

1. Search [existing issues](https://github.com/abdelrzz9/motarjim/issues) to avoid duplicates
2. If no existing issue matches, [open a new issue](https://github.com/abdelrzz9/motarjim/issues/new)

### Discussions

For questions, ideas, or general discussion:

- Open a GitHub Discussion
- Ask in issue comments

## Before Asking

1. Read the [FAQ](docs/faq.md) — your question may already be answered
2. Check the [Troubleshooting Guide](docs/troubleshooting.md) — common issues are documented
3. Search [existing issues](https://github.com/abdelrzz9/motarjim/issues) — someone may have encountered the same problem

## Support Scope

### What We Can Help With

- Bug reproduction and verification
- Clarifying documentation
- Feature requests and design discussions
- Pull request reviews

### What We Can't Help With

- Platform-specific build issues (Flutter, Android, Xcode)
- HTML/CSS design questions unrelated to the compiler
- Production deployment architecture
